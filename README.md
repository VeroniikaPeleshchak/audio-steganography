# 🎧 Audio Steganography: LSB & DWT-SVD

Веб-додаток для стеганографічного приховування тексту в аудіофайлах формату WAV. Розроблено як клієнт-серверний програмний комплекс для дослідження стійкості алгоритмів до шумових та компресійних атак.

## 🚀 Основний функціонал
* **Алгоритми вбудовування:** просторовий (LSB) та гібридний частотний (DWT-SVD).
* **Завадостійке кодування:** інтегровані алгоритми Хеммінга (для LSB) та Ріда-Соломона (для DWT-SVD) для виправлення пошкоджених бітів.
* **Симулятор атак:** модуль для тестування стійкості водяного знака до MP3-компресії, AWGN (шуму), обрізання та низькочастотної фільтрації.
* **Метрики:** автоматичний розрахунок акустичної прозорості (SNR, PSNR) та цілісності даних (BER, NCC).

## 🛠 Стек технологій
* **Backend:** Python, Django (REST API)
* **Frontend:** Streamlit
* **Обробка сигналів:** SciPy, PyWavelets, NumPy, reedsolo, pydub
* **База даних:** SQLite

## ⚙️ Локальний запуск (Getting Started)

### Попередні вимоги
Переконайтеся, що на вашому комп'ютері встановлено **Python 3.10+** (проєкт тестувався на версії 3.12).

### Крок 1. Клонування та налаштування середовища
1. Клонуйте репозиторій на свій комп'ютер:
   ```bash
   git clone [https://github.com/VeroniikaPeleshchak/audio-steganography.git](https://github.com/VeroniikaPeleshchak/audio-steganography.git)
   cd audio-steganography
   ```
2. Створіть та активуйте віртуальне середовище:
   * **macOS / Linux:**
     ```bash
     python3 -m venv venv
     source venv/bin/activate
     ```
   * **Windows:**
     ```cmd
     python -m venv venv
     venv\Scripts\activate
     ```
3. Встановіть усі необхідні залежності:
   ```bash
   pip install -r requirements.txt
   ```

### Крок 2. Запуск серверної частини (Django)
Бекенд обробляє аудіофайли та виконує алгоритми стеганографії.
Перебуваючи в головній папці проєкту з активованим віртуальним середовищем, виконайте:
```bash
cd backend
python3 manage.py runserver
```
*Сервер запуститься за адресою: http://127.0.0.1:8000/*

### Крок 3. Запуск клієнтської частини (Streamlit)
Фронтенд забезпечує зручний веб-інтерфейс користувача.
Відкрийте **нове вікно термінала**, перейдіть у папку проєкту, активуйте віртуальне середовище (див. Крок 1) та запустіть додаток:
```bash
streamlit run frontend/app.py
```
*Додаток автоматично відкриється у вашому браузері за адресою: http://localhost:8501/*
